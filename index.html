Introduction

The OfficeUI framework is a rich command presentation system that provides a modern alternative to the classic HTML controls of traditional web applications. Similar in functionality and appearance to the Microsoft Office 2013 Fluent UI, the framework is composed out of various elements.

Minimum requirements

Minimum supported browser(s) Microsoft Internet Explorer 11
Google Chrome 36
Mozilla Firefox 31
Minimum supported Dependencies Font Awesome 4.0.3
jQuery 2.1.1
jQuery UI 1.11.0
