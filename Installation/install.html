Copy the contents of your package

When we've downloaded the package we can start copy the required files to our web application.
This is a quite easy step as there's only folder which needs to be copies.

  1. Copy the folder Resources to the root of your application folder.

When you're not using specific functions of the OfficeUI some files does not need to be copied.
To find out which files you don't see, review the package dependencies table.

Note: It's not needed to copy the folder to the root of your website, but if you don't, you need to change the path in the file(s) where a reference is made to this. Usually this is the HTML folder. You may also choose to replace the dependencies or not include all the dependencies if you don't need them. More information on that in Managing the package contents.
