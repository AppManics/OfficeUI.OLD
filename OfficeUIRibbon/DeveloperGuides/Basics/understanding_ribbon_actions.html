Understanding the various ribbon actions

The ribbon has a variaty of actions that can be placed on a ribbon. In this section you'll learn about all the different actions that can be placed on a ribbon.

The available actions of a Ribbon are the following:

  • Big icon
  • Small icon
  • Icon with menu items

Big icon

A big icon on the Ribbon represents an icon that takes almost the full vertical space of the Ribbon.
When you want to render a Big icon, you can use the following HTML snippet: 

<a href="#">
  <div class="bigicon">
    <img id="newEmail" alt="New email" src="Resources/Icons/NewMailMessage.png" />
    <div class="iconlegend">New<br />Email</div>
  </div>
</a>

Now, what are all those elements which we can find in a Big icon:

Element Description
A href element Provides a way to link a url, or bind an action to the ribbon.
Div wrapper - class bigicon Provides a way to identify this icon as a big icon.
Img element Provides the image to display for this icon.
Icon legend Provides the text that will be showed under this image.

Small icon

A small icon on the Ribbon represents an icon that takes about a third of the vertical space on the ribbon.
At most, 3 small icons can be placed under eachother. 

<a href="#">
  <div class="smallIcon">
    <img id="newEmail" alt="New email" src="Resources/Icons/NewMailMessage.png" />
    <div class="iconlegend">New<br />Email</div>
  </div>
</a>

Now, what are all those elements which we can find in a Small icon:

Element Description
A href element Provides a way to link a url, or bind an action to the ribbon.
Div wrapper - class bigicon Provides a way to identify this icon as a small icon.
Img element Provides the image to display for this icon.
Icon legend Provides the text that will be showed to the right of this image.

Icon with menu items

Both a small and a big icon can be decorated with a menu. This menu will be showed when we click on the element itself.

Note:  In order to be able to use the icon with the menu items, you should include a reference to font-awesome and jQuery UI.
See the Menu section for more information about this topic.

When you want to render a menu on either a big or a small icon, you should adapt your icon so that inside the legend, there is an i element. 
<a href="#">
  <div class="smallIcon">
    <img id="newEmail" alt="New email" src="Resources/Icons/NewMailMessage.png" />
    <div class="iconlegend">
      New<br />Email
      <i class="fa fa-caret-down"></i>
    </div>
    <!-- Menu placed here. -->
  </div>
</a>
