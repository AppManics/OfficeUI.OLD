Creating your first Ribbon application

In this guide, you'll learn how to create a website which supports the OfficeUI Ribbon. Not all the features will be used in this demo, but after you've completed this guide, you'll be able to create a website on which a Ribbon is visible with some elements on it.

Requirements

In order to sucessfully complete this guide, you should be familiar with the following technologies:

  • HTML
  • Css
  • jQuery
Also, there are some dependencies that are required for the OfficeUI Ribbon to work:
  • jQuery

Make sure that you have a copy of the OfficeUI library and all the dependencies ready on your disk before proceeding.

Step 1: Settings up your website

Create a basic HTML document and include the following stylesheets: 

<link rel="stylesheet" type="text/css" href="Resources/Stylesheets/OfficeUI.Core.css" />
<link rel="stylesheet" type="text/css" href="Resources/Stylesheets/Plugins/Ribbon/OfficeUI.Ribbon.css" />
<link rel="stylesheet" type="text/css" 
href="Resources/Stylesheets/Plugins/Ribbon/Styles/OfficeUI.Ribbon.Blue.css" />
Also, the following scripts must be included (this can be done, right before the closing body tag.) 
<script src="Resources/Scripts/jquery-2.1.1.min.js"></script>
<script src="Resources/Scripts/jquery-ui.min.js"></script>
<script src="Resources/Scripts/Plugins/Ribbon/OfficeUI.Ribbon.js"></script>
<script src="Resources/Scripts/OfficeUI.Core.js"></script>

Note:  Change to path to the Css and Javascript files to point to the required files on the filesystem.

Step 2: Create the area of the Ribbon

When you want to use the Ribbon, it's adviseable to have no padding or margin on the body tag, because this will make the ribbon feel incorrect.
When you create the ribbon, you need to define an area in which the ribbon will be placed. This should be an area of which the id is set to "OfficeUI". 

<div id="OfficeUI">
<!-- Area to render the OfficeUI controls. -->
</div>

Step 3: Define a title for your application.

On your application, you might want to include some branding by specifying a title in the Ribbon. This can be done with the following tags right under the OfficeUI node. 

<div class="title">
  <span>OfficeUI Developer Guide</span>
</div>

Step 4: Define the Ribbon area and add some tabs.

Now we're ready to start building a ribbon, but before we create a ribbon area itself, let's define an area in which the Ribbon should be rendered: 

<div id="mainRibbon" class="ribbon">
<!-- Area to render the OfficeUI controls. -->
</div>

Note:  You'll notice that are Ribbon does have an id here. Choose any id you would like, but keep it in mind since we need it to initialize the Ribbon.

Now that the Ribbon area has been defined, we're ready to create some tabs. Creating tabs is done with the following HTML snippet: 
<div class="tabs">
  <ul role="tablist">
    <li role="tab" class="application" id="tabFile">
      <span>File</span>
      <!-- An application tab cannot contain any contents. -->
    </li>
    <li id="tabHome">
      <span>Home</span>
      <!-- Contents of the tab are rendered here. -->
    </li>
  </ul>
</div>

Note:  You'll notice that all the Ribbon tabs have an id here. This is required for the OfficeUI Ribbon Framework.

Inside every tab, right under the <span> tags, you can place the contents of the ribbon.

Step 5: Defining the contents, groups and action groups

After we have the tabs, it's needed to create the contents inside the tab.
The contents area is defined with the following snippet: 

<div class="contents">
</div>

Note:  You should only have 1 single content section per tab.

inside the contents, we can add the different groups for this tab, together with a legend.
To add a new group in a tab with a legend, the following snippet it used: 
<div class="group">
  <!-- action groups are rendered here. -->
  <div class="legend">New</div>
</div>
Within the group, you can define multiple action groups, which isn't anything more that an area in which actions will be placed. 
<div class="icongroup">
<!-- Actions should be rendered here. -->
</div>

Step 6: Adding some actions

Now that we already a ribbon defined, we can render some actions:
You can render various actions, but in this example, we'll be rendering a big icon. 

<a href="#">
  <div class="bigicon">
    <img id="replyAll" alt="Reply all" src="Resources/Icons/ReplyAll.png" />
    <div class="iconlegend">Reply<br/>All</div>
  </div>
</a>

Step 7: Rendering the Ribbon

Once your HTML is completely formatted, you need to call the following javacsript on the bottom of your page in order to make it work.

<script type="text/javascript">
  var ribbon = $("#mainRibbon").Ribbon(); 
  OfficeUICore.Init();
</script>

Note:  Replace mainRibbon with the id of your ribbon.

Step 8: The final result

When you've completed everything above, your HTML should now look like:

<html>
  <head>
    <link rel="stylesheet" type="text/css" href="Resources/Stylesheets/OfficeUI.Core.css" />
    <link rel="stylesheet" type="text/css" href="Resources/Stylesheets/Plugins/Ribbon/OfficeUI.Ribbon.css" />
    <link rel="stylesheet" type="text/css" href="Resources/Stylesheets/Plugins/Ribbon/Styles/OfficeUI.Ribbon.Blue.css" />
  </head>
  <body>
    <div id="OfficeUI">
      <div class="title">
        <span>OfficeUI Developer Guide</span>
      </div>
      <div id="mainRibbon" class="ribbon">
        <div class="tabs">
          <ul role="tablist">
            <li role="tab" class="application" id="tabFile">
              <span>File</span>
            </li>
            <li id="tabHome">
              <span>Home</span>
              <div class="contents">
                <div class="group">
                  <div class="icongroup">
                    <a href="#">
                        <div class="bigicon">
                        <img id="replyAll" alt="Reply all" src="Resources/Icons/ReplyAll.png" />
                        <div class="iconlegend">Reply<br>>All</div>
                      </div>
                    </a>
                  </div>
                  <div class="legend">New</div>
                </div>
              </div>
            </li>
          </ul>
        </div>
      </div>
    </div>
    <script src="Resources/Scripts/jquery-2.1.1.min.js"></script>
    <script src="Resources/Scripts/jquery-ui.min.js"></script>
    <script src="Resources/Scripts/Plugins/Ribbon/OfficeUI.Ribbon.js"></script>
    <script src="Resources/Scripts/OfficeUI.Core.js"></script>

    
    <script type="text/javascript">
      var ribbon = $("#mainRibbon").Ribbon(); 
      OfficeUICore.Init();
    </script>


  </body>
