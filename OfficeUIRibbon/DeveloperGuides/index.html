Ribbon Framework Developer Guides

The topics contained in this section describe specific aspects of the Windows Ribbon framework.

Basics

Understanding the interface

The Ribbon is splitted in several parts (tabs, groups, icon groups and actions). In this guide, you'll learn what those different aspects are and where they come into play.

Creating your first Ribbon application

In this guide, you'll learn how to create a website which supports the OfficeUI Ribbon. Not all the features will be used in this demo, but after you've completed this guide, you'll be able to create a website on which a Ribbon is visible with some elements on it.
