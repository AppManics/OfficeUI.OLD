OfficeUI Ribbon Framework

The ribbon is a command bar that exposes the features of an application through a series of tabs at the top of a website. It it similar in functionality and appearance to the Microsoft Office 2013 Fluent UI. The ribbon provides an intuitive counterpoint to the trial-and-error process of discovery that is quite common on standard web applications. Optimized for efficiency and discoverability, the ribbon facilitates finding, understanding and using commands with a minimum of mouse clicks and keystrokes through a system of standard controls.
