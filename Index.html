OfficeUI Framework

The OfficeUI Framework is a rich command presentation system that provides a modern alternative to the classic HTML controls of thaditional web applications. Similar in functionality and appearance to the Microsoft Office 2013 Fluent UI, the framework is composed out of various elements.

Topic Description
OfficeUI Framework Overview The topics contained in this section explore the fundamentals of the OfficeUI Framework.
Declaring Commands and Controls with OfficeUI Markup The OfficeUI framework can be rendered using HTML to declaratively implement the appearance of an OfficeUI application.

System Requirements

The OfficeUI suite has been developed for use with the following browsers:

  • Microsoft Internet Explorer
  • Google Chrome
  • Mozilla Firefox

Note  It is recommended to always use the latest browsers of the supported ones which are listed above. It does not mean that the OfficeUI suite does not work with olders versions of the browsers but it is not guaranteed by the development team.
