Here's some general information about the OfficeUI release.
The OfficeUI contains various elements:

Ribbon

The ribbon can be viewed on this page, and no additional page is dedicated to the ribbon.

Menu

The menu is included in the Ribbon and can be opened by clicking on icon that's marked with an arrow.

Context menu

Right click anywhere in this area (main_contents_area) in order to open the context menu.

Breadcrumb

The breadcrumbs are on this page visible just under the ribbon.

Modal popup

The modal popup is a hidden element on this page. Open the context menu and select the last item to load it up.

Input elements

They are a work in progress. The button is already visible in the modal popup.
